# Firebase 대체 무료 플랫폼 비교

## 🏆 추천 순위

### 1. Supabase (가장 추천 ⭐⭐⭐⭐⭐)
**무료 티어:**
- **데이터베이스**: PostgreSQL 500MB 저장 용량
- **읽기/쓰기**: **제한 없음** (무제한!)
- **API 요청**: 월 500,000회
- **파일 저장**: 1GB
- **대역폭**: 5GB/월
- **실시간 구독**: 무제한
- **인증**: 무제한 사용자

**장점:**
- ✅ **읽기/쓰기 제한 없음** (가장 큰 장점!)
- ✅ PostgreSQL (관계형 DB) - SQL 사용 가능
- ✅ 오픈소스, 자체 호스팅 가능
- ✅ Firebase와 유사한 API 구조
- ✅ 실시간 구독 지원
- ✅ 자동 생성 REST API
- ✅ Row-Level Security (RLS)

**단점:**
- ❌ 저장 용량 500MB (Firebase 1GB보다 작음)
- ❌ PostgreSQL 학습 필요 (NoSQL보다 복잡)

**마이그레이션 난이도:** 중간 (API 구조가 유사함)

---

### 2. MongoDB Atlas (추천 ⭐⭐⭐⭐)
**무료 티어:**
- **저장 용량**: 512MB
- **읽기/쓰기**: **제한 없음** (무제한!)
- **네트워크 전송**: 제한 없음
- **백업**: 자동 백업 포함

**장점:**
- ✅ **읽기/쓰기 제한 없음**
- ✅ NoSQL (Firestore와 유사)
- ✅ 다중 클라우드 (AWS, Azure, GCP)
- ✅ 트랜잭션 지원

**단점:**
- ❌ 저장 용량 512MB (가장 작음)
- ❌ 공유 리소스 (성능 제한)

**마이그레이션 난이도:** 쉬움 (NoSQL이라 Firestore와 유사)

---

### 3. Turso (SQLite 기반) (추천 ⭐⭐⭐⭐)
**무료 티어:**
- **저장 용량**: 500MB
- **읽기**: **월 10억 회** (거의 무제한!)
- **쓰기**: **월 1억 회** (거의 무제한!)
- **대역폭**: 150GB/월

**장점:**
- ✅ **거의 무제한 읽기/쓰기**
- ✅ 매우 빠른 성능 (SQLite 기반)
- ✅ 엣지 배포 (전 세계)
- ✅ 무료 티어가 매우 넉넉함

**단점:**
- ❌ SQLite (관계형이지만 제한적)
- ❌ 비교적 새로운 서비스

**마이그레이션 난이도:** 중간

---

### 4. PlanetScale (MySQL) (추천 ⭐⭐⭐⭐⭐)
**⚠️ 중요:** PlanetScale의 무료 티어는 2024년 중단되었을 수 있습니다. 최신 정보는 공식 사이트에서 확인하세요.

**과거 무료 티어 (참고용):**
- **저장 용량**: **5GB** (가장 큼! Firebase의 5배!)
- **읽기/쓰기**: **제한 없음** (무제한!)
- **행 수**: 10억 행
- **브랜치**: 1개 (개발/프로덕션 분리)
- **대역폭**: 무제한

**현재 플랜:** 유료 플랜 시작가 $5/월 (단일 노드 Postgres) 또는 $15/월 (고가용성 클러스터)

**장점:**
- ✅ **저장 용량 5GB** (모든 플랫폼 중 가장 큼!)
- ✅ **읽기/쓰기 무제한** (할당량 걱정 없음)
- ✅ MySQL (안정적이고 널리 사용되는 관계형 DB)
- ✅ 브랜치 기능 (Git처럼 개발/프로덕션 분리)
- ✅ 무제한 대역폭
- ✅ 자동 스케일링
- ✅ 무료 백업 포함

**단점:**
- ❌ MySQL 학습 필요 (NoSQL보다 복잡할 수 있음)
- ❌ 무료 티어는 1개 브랜치만 (개발/프로덕션 분리 제한)

**마이그레이션 난이도:** 중간 (관계형 DB라서 스키마 설계 필요)

## 💡 마이그레이션 고려사항

### 현재 앱에서 사용 중인 Firebase 기능:
1. ✅ Firestore (데이터베이스) → 대체 가능
2. ✅ Storage (파일 저장) → 대체 가능
3. ❌ Authentication (인증) → 현재 사용 안 함
4. ❌ Functions (서버리스) → Vercel 사용 중

### 마이그레이션 시 필요한 작업:
1. 데이터베이스 쿼리 코드 수정
2. Storage API 코드 수정
