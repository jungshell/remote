- **2** 오늘과 비슷한 날 추천 **(완료)**
- **3** 일기 수정/편집 **(완료)**
- **4** 사진 앨범 뷰 **(완료)**
- **6** 월별/연도별 요약 리포트 자동 생성
- **7** 장소별 방문 히스토리 맵

- **새 아이디어 1** 해시태그 클릭 시 연관 일기 목록
- **디자인·이용 편의성 1** 모바일 터치 영역 확대 (탭/버튼)
- **디자인·이용 편의성 2** 로딩 상태 시각화 (스피너/스켈레톤)
- **디자인·이용 편의성 3** 삭제 확인 인라인 UI (confirm 대신)
- **디자인·이용 편의성 4** 에러/성공 메시지 토스트(스낵바)
- **디자인·이용 편의성 5** 일기 목록 가독성·스크롤 (날짜 그룹핑, 맨 위로)

## 감각적·직관적 디자인 방향 5가지 (참고 사이트 + 우선순위)

우선순위는 1 → 5 순 (1이 가장 추천).

| 순위 | 방향 | 한 줄 설명 | 참고 사이트 |
|------|------|------------|-------------|
| **1** | 감성·워밍 톤 | 따뜻한 색감, 둥근 카드, 넉넉한 여백으로 ‘가족 일기’ 감성 강화 | [Day One](https://dayoneapp.com) |
| **2** | 미니멀 블록/카드 UI | 일기·사진·키워드를 블록 단위로 구분해 스캔·이해가 쉬움 | [Notion](https://notion.so) |
| **3** | 타임라인 시각화 | 연대기처럼 ‘시간의 흐름’이 보이는 레이아웃 | [Apple Photos - 추억](https://www.apple.com/kr/iphone/features/photos/) |
| **4** | 마이크로 인터랙션 | 버튼·카드 전환·로딩에 작은 애니메이션으로 만족도 향상 | [Linear](https://linear.app) |

## 고도화 아이디어 10개

## 1. AI 기반 감정 트렌드 분석 📊
- **기능**: 월별/연도별 감정 점수 그래프
- **효과**: 가족의 기분 변화 패턴 파악
- **구현**: Chart.js + moodScore 데이터 시각화

## 2. 음성 녹음 품질 자동 개선 🎙️
- **기능**: 노이즈 제거, 음성 향상
- **효과**: 인식 정확도 향상
- **구현**: Web Audio API + 노이즈 제거 알고리즘

## 3. 가족 구성원별 개인화된 대시보드 👥
- **기능**: 각 구성원별 전용 일기 뷰
- **효과**: 개인 맞춤 경험
- **구현**: 필터링 + 개인화 레이아웃

## 4. 자동 타임라인 생성 📅
- **기능**: 여러 일기를 연결한 연대기
- **효과**: 가족의 성장 스토리 시각화
- **구현**: 날짜 기반 정렬 + 스토리텔링

## 5. 음성 메모 자동 분류 🗂️
- **기능**: 주제별 자동 태깅 (학교, 놀이, 여행 등)
- **효과**: 검색 및 필터링 용이
- **구현**: LLM 기반 주제 분류

## 6. 가족 일기 통계 대시보드 📈
- **기능**: 가장 많이 나온 키워드, 자주 가는 장소 등
- **효과**: 가족 패턴 인사이트
- **구현**: 데이터 집계 + 시각화

## 7. 자동 백업 및 복원 🔄
- **기능**: 정기적 자동 백업 + 복원 기능
- **효과**: 데이터 손실 방지
- **구현**: Firebase Export/Import

## 8. 다국어 지원 🌍
- **기능**: 영어, 일본어 등 자동 번역
- **효과**: 해외 거주 가족 지원
- **구현**: Google Translate API

## 9. 소셜 공유 최적화 📱
- **기능**: 인스타그램/카카오톡 스토리용 이미지 자동 생성
- **효과**: 쉬운 공유
- **구현**: 이미지 템플릿 + 자동 리사이징

## 10. AI 기반 일기 추천 시스템 💡
- **기능**: "오늘과 비슷한 날" 추천
- **효과**: 과거 추억 자동 연결
- **구현**: 유사도 알고리즘 + 키워드 매칭
