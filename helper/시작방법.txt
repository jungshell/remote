※ 실행하기.bat 안에는 한글이 없습니다. (검은 창에서 글자가 깨지던 문제 방지)
※ 웹 주소는 보통 http://localhost:5188 입니다. (5173 아님)
  이전에 켜 둔 웹 창이 있으면 5188이 이미 쓰일 수 있어요. 그땐 Vite 검은 창에 적힌
  "Local: http://localhost:숫자" 를 브라우저에 입력하세요.
※ apps/web 폴더에 .env.local 을 예전에 만들었으면 삭제하세요. (8000 같은 잘못된 주소가 들어 있으면 업로드가 안 됩니다)
※ API 창을 예전 방식(router 없이)으로 켠 적 있으면 모두 닫고, 실행하기.bat 을 다시 실행하세요.

①  "실행하기.bat" 파일을 더블클릭합니다.
    (검은 창 두 개가 뜨면 정상입니다. 둘 다 닫지 말고 그대로 두세요.)

②  잠시 후 브라우저가 열립니다.
    안 열리면 주소창에 아래를 직접 입력하세요.

        http://localhost:5188

③  로그인
    이메일: admin@cnci.local
    비밀번호: admin1234

④  위에서 "회의/STT", "규정 Q&A" 같은 메뉴를 눌러 씁니다.


--------------------------------------------------------------------------------
  헷갈릴 때
--------------------------------------------------------------------------------

• 화면에 빨간 글씨가 나오면 → 맨 위 "실행하기.bat"으로 다시 켠 뒤 새로고침.

• "규정 Q&A"에 PDF를 넣었는데 안 되면
  → PDF가 사진만 있는 스캔본일 수 있습니다. 그럴 땐 글자가 있는 파일로 바꾸거나,
    다른 프로그램으로 글자를 뽑아서 "규정 원문" 칸에 붙여 넣고 등록하세요.

• 규정 질의(채팅) 답변이 안 나오면
  → Gemini 를 쓰려면 services\api\gemini_key.txt 파일을 만들고 API 키를 한 줄만 넣으세요.
    실행하기.bat 가 이 파일을 읽어 CNCI-API 창에 넘깁니다. (키 파일은 다른 사람과 공유하지 마세요.)
  → 키가 없으면 PC에 Ollama( localhost:11434 )가 떠 있어야 하며, AI_PROVIDER=ollama 입니다.

• 같은 PDF를 여러 번 올려 제목이 중복됐다면
  → "규정 Q&A"에서 「문서 등록·관리 펼치기」→「제목 중복 정리」로 가장 먼저 등록한 1건만 남길 수 있습니다.
  → 이후에는 같은 파일명(문서 제목)이 있으면 덮어쓸지 묻는 창이 뜹니다.

• 컴퓨터를 껐다 켰으면 → 다시 "실행하기.bat"을 실행해야 합니다.


--------------------------------------------------------------------------------
  처음 한 번만 (실행하기가 안 될 때)
--------------------------------------------------------------------------------

• PHP 설치: https://windows.php.net/ 에서 받아서 설치하거나,
  이미 설치했다면 명령 입력창에서  php -v  가 나오는지 확인.

• 규정 PDF 등록 시 서버 오류가 나면: php.ini 에서  extension=mbstring  줄의 앞 ; 을 제거해 켜 주세요.
  (최신 프로그램은 mbstring 없어도 일부 동작하지만, PHP 자체에 켜 두는 것이 가장 안정적입니다.)

• Node.js 설치: https://nodejs.org/ 에서 "LTS" 받아 설치.
  설치 후 명령 입력창에서  node -v ,  npm -v  가 나오는지 확인.

• 이 폴더(helper)에서 명령 입력창을 열고 한 번 실행:
    npm install
