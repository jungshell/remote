# 답변 정확도·정합성 향상 방안

충콘지니 답변의 정확도와 정합성을 높이기 위한 방안을 정리했습니다.

---

## 1. 적용된 사항 (코드 반영)

| 항목 | 내용 |
|------|------|
| **제규정·법령 조문 표기** | 제규정·법령 인용 시 반드시 조문(제○조, 제○항) 표기. 법령은 개정일자까지 표기. |
| **제규정 우선 + 종합의견 구조** | 1차로 **제규정**을 상단에 검토해 의견 제시. **상위 법령과 상충** 시에만 하단에 **종합의견** 추가. 상응 시 제규정만으로 의견 마무리. |
| **참고 문서 없으면 추측 금지** | 등록된 문서에 없는 내용은 "담당 부서에 확인해 주세요" 등으로 답변. |
| **담당 부서 연계** | 규정·법령 해석이 애매할 때 "담당 부서(총무·인사·해당 업무 담당) 확인" 명시. `CONTACT_DEPARTMENT` 상수로 정리. |
| **참고 출처 자동 표기** | 답변 말미에 참고한 문서명·페이지를 붙여 사용자가 원문 검증 가능. |
| **검색 문맥 확대** | 참고 문서를 3개 → 5개로 늘려 관련 조문·절차를 더 많이 참고. |
| **생성 옵션** | `temperature` 0.35 등으로 낮춰 사실 중심·일관된 답변 유도. |
| **사용자 피드백 수집** | 👍/👎 도움됨·부족함 + **🚩 오답 신고** 버튼. 오답 신고는 `answer_reports` 컬렉션에 저장 후 관리자 탭에서 검토. |
| **법령 DB 연동 스텁** | `getExternalLawContext(query)` 함수 추가. 추후 국가법령정보센터 등 API 연동 시 여기서 조회해 프롬프트에 반영 가능. |

---

## 2. 지식 베이스 품질 (관리자 권장)

- **제규정 업로드 시**
  - 조문 번호가 포함된 본문을 올리면, AI가 "제○조", "제○항"을 그대로 인용할 수 있습니다.
  - 규정명·개정일·시행일이 문서 상단에 있으면 법령 인용 시 개정일자 표기에 도움이 됩니다.
- **카테고리 구분**
  - "진흥원 규정" / "e나라도움" / "사업 공고" 등으로 나누어 두면, 범주별로 더 정확히 검색됩니다.
- **중복·구버전 정리**
  - 개정된 규정이 있으면 구버전은 삭제하거나, 제목에 "(폐지)" 등 표기해 혼선을 줄입니다.

---

## 3. 사용자 활용 방법

- **범주 선택**
  - 규정 질문 시 사이드바에서 "진흥원 규정"을 선택하면 해당 문서 위주로 답변합니다.
- **질문 구체화**
  - "출장비 절차"보다 "출장 여비 지급 절차와 제출 서류"처럼 구체적으로 묻으면, 절차·양식·조문이 더 정확히 나옵니다.
- **출처 확인**
  - 답변 하단 "참고 출처"의 문서명·페이지로 원문을 확인해 보시면 정합성 검증에 도움이 됩니다.

## 4. 추가로 고려할 수 있는 방안

| 방안 | 상태 |

| **사용자 피드백 수집** | ✅ 적용 (👍/👎 + 오답 신고, 관리자 탭에서 검토) |
