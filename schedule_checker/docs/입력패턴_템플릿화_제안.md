# 입력 패턴 기억 → 템플릿화로 업무 효율 높이기

사용자가 반복해서 기입하는 패턴을 **기억했다가 나중에 템플릿으로 제안**하면, 할 일 추가·정리 시간을 줄이고 업무 효율을 높일 수 있습니다.

---

## 1. 어떤 패턴을 기억할지

| 패턴 | 설명 | 활용 예 |
|------|------|----------|
| **제목 유사도** | 비슷한 제목으로 자주 추가하는 할 일 | "주간 회의 준비", "월간 보고서 작성" → 주/월 단위 반복 제안 |
| **마감 시간대** | 자주 선택하는 마감 시각(예: 18:00, 9:00) | 오후 6시 마감이 많으면 기본값을 18:00로 제안 |
| **담당자** | 특정 담당자에게 자주 배정 | "홍길동" 선택 비율이 높으면 담당자 드롭다운 상단에 노출 |
| **목표–세부 조합** | 같은 상위 목표 아래 비슷한 세부 업무를 반복 추가 | "Q1 마케팅" 아래 "기획→디자인→검수" 패턴 → 한 번에 세부 3개 제안 |
| **장소·참석** | 특정 장소·참석자 조합이 반복 | "회의실 A", "김팀장, 이대리"가 자주 쓰이면 자동완성 또는 추천 |
| **우선순위** | 특정 유형의 할 일에 항상 높음/낮음 선택 | "긴급"이 들어간 제목은 우선순위 높음으로 제안 |

---

## 2. 템플릿화 제안 방식

### 2.1 자동 제안(암묵적)
- **할 일 추가 모달**을 열 때: 지난 N개 할 일의 **마감 시간·담당자·우선순위** 분포를 보고, 가장 많이 쓰인 값을 **기본값**으로 설정.
- **세부 업무 추가** 시: 해당 메인 업무의 기존 세부 업무 **마감일 패턴**(예: 메인 마감 1주 전)을 적용한 일시 제안.

### 2.2 명시적 템플릿
- **저장**: 사용자가 "이걸 템플릿으로 저장"을 선택하면, 현재 할 일(또는 메인+세부 묶음)의 **제목·설명·마감 규칙·담당자·우선순위·세부 목록**을 템플릿으로 저장.
- **불러오기**: 할 일 추가 시 "템플릿에서 가져오기"에서 목록을 고르면, 해당 템플릿의 필드가 폼에 채워짐. 날짜만 "오늘/내일" 등으로 재계산.

### 2.3 패턴 기반 제안
- **제목 입력 중**: 과거 제목과 유사한 문자열이 있으면 **자동완성** 또는 "비슷한 할 일: ○○ (마감 3/20)" 형태로 하단에 노출. 선택 시 해당 할 일의 **마감·담당자·설명**을 가져와 폼에 채움.
- **주/월 반복 감지**: "주간 회의", "월간 리포트"처럼 같은 제목이 주기적으로 추가되면, "이 할 일을 **매주/매월 반복**으로 등록할까요?" 안내 후 반복 할 일로 전환 제안.

---

## 3. 구현 시 고려 사항

| 항목 | 내용 |
|------|------|
| **저장 위치** | 템플릿·패턴 통계는 사용자별로 저장 (Firestore `users/{uid}/templates`, `users/{uid}/taskPatterns` 등). |
| **개인정보** | 패턴 분석은 모두 해당 사용자 디바이스 또는 본인 데이터에만 적용. 타인과 공유하지 않음. |
| **초기값** | 패턴 데이터가 부족할 때는 현재처럼 **일시 = 지금에 가장 가까운 30분**, **담당자 = 미지정** 등 안전한 기본값 유지. |
| **덮어쓰기** | 템플릿/자동완성으로 채운 값은 사용자가 수정 가능. "제안만 하고 선택은 사용자"가 원칙. |

---

## 4. 단계별 도입 제안

| 단계 | 내용 | 효과 |
|------|------|------|
| **1단계** | 할 일 추가 시 **마감 시간·담당자**의 "가장 최근/가장 많이 쓴 값"을 기본값으로 사용 | 클릭 수 감소 |
| **2단계** | "템플릿으로 저장" / "템플릿에서 불러오기" UI 추가 (할 일 1건 또는 메인+세부 묶음) | 반복 업무 입력 시간 단축 |
| **3단계** | 제목 **자동완성** 또는 "비슷한 과거 할 일" 제안 | 실수 방지, 이전 설정 재사용 |
| **4단계** | 반복 패턴 감지 후 "매주/매월 반복 할 일" 전환 제안 | 정기 업무 자동화 |

이 순서로 적용하면, **기입 패턴을 기억 → 템플릿·기본값 제안**으로 업무 효율을 단계적으로 높일 수 있습니다.
